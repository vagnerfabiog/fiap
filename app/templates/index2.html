API de Consultas - Embrapa

Consultas a dados de vitivinicultura da Embrapa

Sobre o Projeto

Este projeto é parte do TECH CHALLENGE da fase 1 da Pós Tech da FIAP do curso Machine Learning Engineering.

A API permite consultas a dados relevantes sobre a produção, processamento, comercialização, importação e exportação relacionados ao setor vitivinícola.

Utilização

Você pode acessar a API através das rotas definidas para obter informações específicas sobre a vitivinicultura.
